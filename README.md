# NJUCS-exercise

NJUCS配套的刷题软件

因为会有公式，为了保证显示效果，推荐pdf导出

软件需要的数据库在release中

## 功能

1、题库数据增删改查；

2、刷题练习；

3、数据分析；

4、题目去重；

5、导出为其他格式文件，如PDF、docx等；

6、提供题目搜索过滤功能；

7、提供填空、选择、解答、判断等题型；

8、正确判断；

文字处理采用HTML格式文本

使用最长句子MD5值进行重复判断

# 配置

OS: Manjaro Linux 18.1.1

Qt: 5.13.1

GCC: 9.2.0

C++: C11

SQL: sqlite3
